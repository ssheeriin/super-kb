# Support

## Where to Get Help

- Usage questions: open a GitHub issue and describe what you are trying to do
- Bug reports: use the bug report issue template
- Feature requests: use the feature request issue template
- Security issues: follow [SECURITY.md](SECURITY.md)

## What to Include

For faster help, include:

- SKB version: `skb-mcp-server version`
- Operating system
- Install method: installer script, wheel, or source checkout
- Claude Code version: `claude --version`
- Relevant output from `skb-mcp-server doctor`
- Reproduction steps and expected behavior
